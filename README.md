## 🚀 Getting Started

Each subdirectory contains its own Dockerfile and/or docker-compose configuration, along with specific documentation for building and running that particular image. Navigate to the service you're interested in and follow the instructions in its README.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

## 📦 Available Images

Each image in this repository includes:
- Detailed README with usage instructions
- Dockerfile for building the image
- docker-compose.yml (where applicable)
- Any necessary configuration files

<p align="right">(<a href="#readme-top">back to top</a>)</p>

## 🔄 Updates and Maintenance

Images are only updated to include features that I need personally. However, if you find an issue with something here, please start a new issue on the repo so I can fix it and everyone can benefit.

These changes include:

- Security patches
- Base image updates
- Bug fixes
